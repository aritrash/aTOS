# aTOS

aTOS is a lightweight embedded operating system developed for modern microcontrollers.

Designed as the firmware counterpart to **aTerm**, it provides a modular architecture consisting of:

- Hardware Abstraction Layer (HAL)
- Bootloader (aTLoader)
- Kernel
- Shell
- Driver Framework
- Platform Layer
- Custom serial protocol

## Current Target

- ESP32 DevKit V1
- Arduino Framework
- PlatformIO

## Companion Project

aTerm is the official desktop terminal for aTOS.
